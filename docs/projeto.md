# Descrição do Projeto

## Sistema de Gerenciamento de Biblioteca

Sistema completo desenvolvido para a disciplina CC8550 - Simulação e Teste de Software.

### Objetivo

Desenvolver um sistema de gerenciamento de biblioteca com arquitetura modular, separação de responsabilidades e cobertura completa de testes.

### Funcionalidades

- Gerenciamento de Livros, Usuários, Empréstimos, Autores e Categorias
- Operações CRUD completas
- Regras de negócio complexas
- Interface CLI interativa
- API REST completa
- Sistema de validação robusto
- Tratamento de exceções personalizado

### Tecnologias

- Python 3.8+
- FastAPI
- SQLAlchemy
- SQLite
- Pytest
- Mutmut
